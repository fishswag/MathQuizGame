# General Math & Calculus Quiz 🎮

A polished Python GUI quiz game built with Tkinter featuring random math and calculus questions, multiple difficulty levels, and a clean cozy interface.

---

## ✨ Features

* 🎯 Difficulty levels (Easy, Medium, Hard)
* 📚 Categories (Arithmetic, Algebra, Calculus, Mixed)
* ❤️ Lives system
* 🔥 Streak bonus (+1 every 3 correct)
* ⏱️ Timer with total time tracking
* 📊 Progress bar
* 🔀 Random question generation (no hardcoding)
* 🎲 Multiple choice + typed answer modes
* 🧠 Question history (reduces repeats)
* 🎨 Clean and cozy UI design
